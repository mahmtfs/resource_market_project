About Page

This is Resource Market, a python university project.
The project is a website made with Flask framework.
Databases are handled by SQLAlchemy module.
The website has the following functionality:
1) Create, Read, Update and Delete for users and items.
2) User authentication.
3) Buy/Sell system with balance and item depletion.
4) Search.
5) Pagination.
